# lmr-corretora-landing-page
Modern landing page for LMR Planos de Saúde built with Next.js, React, TypeScript, Tailwind CSS and shadcn/ui.

# LMR Planos de Saúde

Landing page institucional desenvolvida para a LMR Corretora de Seguros.

## Tecnologias utilizadas

- React
- Next.js
- TypeScript
- Tailwind CSS
- shadcn/ui
- Lucide Icons

## Funcionalidades

- Landing page responsiva
- Integração com WhatsApp
- Sessão de parceiros
- CTA para conversão
- Layout moderno e otimizado
- SEO básico com Next.js
